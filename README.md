# Detección de Phishing en Argentina – DiploDatos 2025

Proyecto de Ciencia de Datos desarrollado en el marco de la Diplomatura en Ciencia de Datos (FAMAF, 2025).

Inspirado en el Robo al Banco Río de 2006, el proyecto aborda su versión moderna: el robo de datos a través de sitios web fraudulentos en Argentina.

## Objetivo
Desarrollar una solución integral para detectar sitios web de phishing y fraude online mediante:
- Web scraping
- Análisis exploratorio de datos
- Ingeniería de características
- Modelos de Machine Learning supervisados

## Metodología
- Recolección y scraping de sitios web
- Curación y normalización de datos
- Análisis exploratorio (EDA)
- Entrenamiento y evaluación de modelos
- Despliegue de una aplicación web interactiva

## Tecnologías
Python · Pandas · NumPy · Scikit-learn · BeautifulSoup · Matplotlib · Streamlit
